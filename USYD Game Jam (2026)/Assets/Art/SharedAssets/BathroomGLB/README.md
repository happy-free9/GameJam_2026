# Shared Bathroom GLB Assets

Reusable bathroom environment and prop assets from the RenoPilot Bathroom Designer project.

- Format: `.glb`
- Intended use: hotel, bathroom, corridor, or prop scenes.
- Team members may select only the assets needed for their own scene.
- Do not rename a model without checking whether another scene or prefab references it.
- Unity may require glTFast or another glTF importer before these assets can be used as Unity prefabs.
- After importing, check scale, orientation, materials, transparency, lighting, and colliders.
